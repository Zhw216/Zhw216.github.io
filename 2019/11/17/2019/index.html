Csp-S 2019 算是总结吧，

发表于 2019-11-17

考完Day2后，同学们的心情都倍现压抑。
第一届Csp-S，对于我们来说，题出的挺难的。
考的都是一些令人一眼看完不知何从下手的算法，平常学习的数据结构仿佛空壳一般无用。
CSP试题褪去了Noip之前的略带趣味，变得更为专业，难度也明显提高。
FHOI最有希望的两个高一新生，可能要抱着差强人意的成绩回到这个校园，
确实，这令人很失望。对于我们，内心的感觉无非就是歉疚和感伤。
高二学长纷纷退役，给我们留下的是言语上的鼓励和心灵上的支持。
面对他们，我仿佛看见了以后的自己。
都说长江后浪推前浪，我们这一届新高一的浪仿佛蔫不拉几的。
面对如此境遇，谁都笑不出来。
现在留给我们的路有两条：
放弃学OI，好好学习文化课；
继续学OI，明年再去参加CSP。
我选择继续OI生涯，而我的一位好友——lxy，似乎已经下定决心要退役了。
平常热闹欢快的机房，此时变得如此凄清。
平时都嫌话多的三人，现在却一个字都说不出来。
看着电脑屏幕上码了数遍的各种模板，哭笑不得。
转头望去，
后黑板上还留着我亲笔写的“CSP-S 2019 RP++”；
前黑板充满了我们做套题时打的草稿和笔记，
现在看来，倒像是冷嘲热讽一般。
开着暖风空调，依旧驱逐不了一个OIER心中的寒意。
直入心底的那种。
竞赛的失利，使未来的路，又添了几分渺茫。
呆坐在电脑前，不知所措。
就连我潜意识里都觉得，我快要不行了。
但我是个不服输的人，毕竟无奋斗，不青春。
我还想再搏最后一把。
为了这无悔的青春，
都说年轻是资本，
为什么不来一票大的呢？
CSP-S 2019，已成定局。
CSP-S 2020，等我归来。

——_zhw
